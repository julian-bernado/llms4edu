# slides for the ei fellows workshop

## workshop website

The temporary workshop landing page lives in [`docs/`](docs/). It is a Jekyll
site using GitHub Pages' `minima` theme.

To publish it, configure GitHub Pages to deploy from the `docs` folder on the
default branch. Event details and the two resource destinations are maintained
in [`docs/_config.yml`](docs/_config.yml). Until `interactive_url` and
`slides_url` are filled in, the homepage presents them as unavailable rather
than sending visitors to dead links.
